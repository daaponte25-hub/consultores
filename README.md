# Consultores | Web Profesional

Este proyecto es una plataforma web profesional y responsive, diseñada para ofrecer una experiencia de usuario rápida y moderna. Combina el rendimiento de **React** y **Vite** con la potencia de gestión de contenido de **WordPress** a través de su API REST.

## 🚀 Características Principales

*   **Arquitectura Headless:** WordPress actúa como el backend (CMS) para la gestión de contenido, mientras que React proporciona una interfaz de usuario dinámica y rápida.
*   **Blog Dinámico:** Implementación completa de CRUD (Crear, Leer, Actualizar, Borrar) utilizando la API REST de WordPress.
*   **Diseño Responsive:** Interfaz moderna y adaptable a cualquier dispositivo gracias a **Tailwind CSS**.
*   **Optimización:** Desarrollo rápido y configuración optimizada con **Vite**.

## 🛠 Tecnologías Utilizadas

*   **Frontend:** React, Vite.
*   **Estilos:** Tailwind CSS.
*   **Backend/CMS:** WordPress (vía API REST).
*   **Lenguaje:** JavaScript / JSX, css.
